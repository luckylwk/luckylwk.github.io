K-MEANS CLUSTERING ALGORITHM (D3)

Basic implementation of the K-Means clustering algorithm. Uses Voronoi Tessalation mapping to display the cluster area's.

Performance of the clustering algorithm is heavily dependent on the location of both the randomized pivots (used for generation of the datapoints) and the randomized cluster pivots.

SORTABLE BAR-CHART (D3)

Bar chart that has the ability to switch (animated) between two different datasets. It also has the option to toggle between sorting the dataset on its labels (x-axis) and the values. The value-sorting is now based on the sum of the two values per x-label.
